# agentic-governance

Agent plugin containing governance and customization-audit assets from the agentic workspace.

## Included asset sets

- Skills: audit-* and governance-* skills plus orchestration helpers.
- Agents: governance and review-focused agents.
- Instructions: governance and supporting policy instructions.
- Prompts: governance and audit prompts.

## Install as local plugin location

Add this plugin directory to VS Code settings:

```json
"chat.pluginLocations": {
  "c:/Projects/vs_local_marketplace/plugins/agentic-governance": true
}
```

## Marketplace entry

This plugin is also indexed by local marketplace metadata in `.github/plugin/marketplace.json`.
