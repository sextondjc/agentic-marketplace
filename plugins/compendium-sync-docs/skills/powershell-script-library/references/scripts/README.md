# Niche Procedures Scripts

Store niche, reusable PowerShell procedures here when they are too specific for `.github/scripts/powershell` but still needed by this skill.

## Rules

- Prefer shared promotion first; keep local only when scope is skill-specific.
- Each script must include a brief header with purpose, parameters, and expected output.
- Add or update a reference from this folder in `SKILL.md` when introducing a new script.
- Re-evaluate promotion to shared catalog when a script is used by multiple skills.
