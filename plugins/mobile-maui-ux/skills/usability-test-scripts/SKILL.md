# Usability Testing Scripts

## Specialization

Create structured usability test scripts and scoring materials for mobile flows, including accessibility observations.

## Trigger Conditions

- A prototype or app flow must be validated with participants.
- Team needs severity-ranked findings to prioritize fixes.

## Inputs

- Test objective and target flows.
- Participant criteria and device matrix.
- Prototype/app build and moderation constraints.

## Required Outputs

- Participant screener and consent script.
- Task scripts with neutral prompts and bail-out rules.
- Completion and severity rubric.
- Findings template and prioritized remediation backlog.

## Workflow

1. Define objective and measurable success criteria.
2. Build participant screener and session setup script.
3. Draft 3-7 task scripts across core and error flows.
4. Add scoring rubric and accessibility checklist.
5. Run sessions, aggregate findings, and prioritize actions.

## Self-Containment

This workflow is self-contained and includes planning, facilitation, scoring, and prioritization within this skill.

## Done Criteria

- Scripts are neutral, repeatable, and measurable.
- Severity model is explicit and actionable.
- Findings template supports direct prioritization.
